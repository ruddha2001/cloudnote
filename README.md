# Usage

<!-- usage -->

```sh-session
$ npm install -g cloudnote
$ cloudnote COMMAND
running command...
$ cloudnote (-v|--version|version)
cloudnote/0.0.0 win32-x64 node-v12.18.0
$ cloudnote --help [COMMAND]
USAGE
  $ cloudnote COMMAND
...
```
